## Sizing

Crop browser chrome / URL bar before committing. Aim for < 500 KB per PNG. Run
`oxipng -o4 *.png` or convert to WebP (`cwebp -q 80 in.png -o out.webp`) for
smaller files; if you switch to WebP, update the references in the project
README accordingly.
